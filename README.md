# Example API in Go

This is a simple API developed in Go (Golang) for learning purposes. The API allows performing basic CRUD (Create, Read, Update, Delete) operations on a fictional dataset.

## Features

- Store all movies
- List all movies
- Update information of an existing movie
- Delete a movie

## Technologies Used

- Go (Golang)
- [net/http](https://pkg.go.dev/net/http) for handling HTTP requests
- [encoding/json](https://pkg.go.dev/encoding/json) for JSON manipulation
- [gorilla/mux](https://github.com/gorilla/mux) for routing

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/your-repository.git
   cd your-repository
2. Install Dependecies

    ```bash
    go mod tidy
3. Execute

    ```bash
    go run cmd/main.go
